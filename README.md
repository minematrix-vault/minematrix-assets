# MineMatrix Assets CDN

This repository hosts the heavy assets for MineMatrix (https://minematrix.web.app).

## Usage via jsDelivr
To link an asset in the website code, use:
`https://cdn.jsdelivr.net/gh/[GITHUB_USERNAME]/[REPO_NAME]@[VERSION]/[PATH_TO_FILE]`

### Folders:
- `/packs`: Minecraft Resource and Texture packs.
- `/overlays`: High-resolution video overlays.
- `/shaders`: Shader packs and configuration files.
- `/thumbnails`: High-CTR thumbnail templates.
